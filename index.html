🔗A/Hチーム共通課題
  • 📂 データ・ノウハウの散在・質のバラツキ
    • 例：必要なデータがPEAS、BOX、NASなど複数箇所に保存されていて探すのに時間がかかる
    • 例：過去の装置トラブル対応履歴が各GrのExcelやメールにしか残っておらず、担当者が異動すると情報が消える
  • 👨‍🏫 技術伝承・教育不足・属人化
    • 例：アラーム発生時、知見者に個別に問い合わせるしかなく、同じ問い合わせが何度も発生
    • 例：教育コンテンツや履歴の未整備、若手のローテーション時に引継ぎ工数が膨大
  • 🛠️ システム・ツールの使い勝手や統合
    • 例：複数ツールの乱立、どれが便利か分からない、情報アクセスが非効率
    • 例：自動測長やレシピ作成ツールなどが多様化し、使いこなせていない
  • 🤖 生成AI等の新技術活用
    • 例：ChatGPT以外のAIが使えない、現場でのAI活用が進まない
    • 例：ノーコード/ローコードAIツールを現場で活用できず、非エンジニアの業務効率化が進まない
  • 🔄 業務プロセスの非効率・見直し
    • 例：無駄な会議や承認、資料作成が多い
    • 例：設計以外の業務が多く本来業務に集中できない

  • 🏗️ 設計・開発プロセスの負担、組織運営

  • 🔒 情報管理のサイロ化・意思決定の遅延
    • 例：各部門が個別に情報を管理しているため、データの断片化が発生
  • 🌐 他業界との情報交換・技術交流

  • 📦 MASTER BOMやパーツ管理のシステム化

課題1：データ・ノウハウの一元管理と質の標準化
📂As is（現状）
  • データ・ノウハウが複数システムやExcel、メール等に分散
  • 必要な情報がすぐに見つからない、検索性が低い
  • データの質やフォーマットがバラバラ
  • ノウハウが個人や部門ごとに管理され、全社的な共有・標準化が進んでいない
  • 過去事例は「前任者に聞く」ことでしか得られない
📂To be（あるべき姿）
  • 全社横断で一元管理されたデータベースを構築
  • データ・ノウハウの標準化・統一フォーマット化
  • 誰でも必要な情報に素早くアクセス・検索できる
  • ナレッジの統合DB化・アクセス性の向上
  • 過去事例やノウハウが即座に参照できる仕組み
知識循環型エンジニアリングフローの導入
🔄知識循環型エンジニアリングフローの仕組み
  • 設計・評価・トラブル・顧客フィードバックなど全ての技術情報を一元的に収集・蓄積
  • RAGシステム（例：ちけんの杜、NotebookLM）を活用し、幅広い知識と専門知識を循環的に蓄積
  • FMEALibrary等の専用DBでトラブル事例や設計根拠を体系的に管理
  • 設計ガイドライン管理システムで標準化・動的な設計指針を提供
  • 全ての情報が次の開発案件に活かされる「知識循環」を実現
1設計・評価・トラブル情報をDB化
→
2RAGシステムで知識を循環・分析
→
3設計ガイドライン・ナレッジベースへ反映
→
4次案件で即活用・継続的進化
仕組み例:
・設計プロセスで得た知見を「ちけんの杜」や「FMEALibrary」に自動蓄積
・全社横断で検索・参照できるナレッジベースを構築
・設計ガイドラインはRAGシステムで常に最新化・標準化
・過去の失敗・成功事例も含めて次の開発に活かす

課題2：システム・ツールの統一・自動化
🛠️As is（現状）
  • 複数ツール・システムが乱立し、使い分けが煩雑
  • Excelやマクロによる手作業が多い
  • システム間の連携がなく、情報の転記や重複作業が発生
  • 自動化・IT化が進まず、業務効率が低い
  • AIやノーコード/ローコードツールの活用が限定的
🛠️To be（あるべき姿）
  • システム・ツールの統一・一本化（全社横断の業務プラットフォーム）
  • Web・専用システムやAPIによる自動化推進
  • システム間連携によるデータの自動流通・転記不要
  • AIやノーコード/ローコードツールの現場活用
  • 業務フローの標準化・効率化（申請・承認・資料作成の自動化）
  • ユーザーごとに最適化されたダッシュボード・ワンストップUI
全体最適化・自動化の仕組み
🔗統合業務プラットフォーム
  • 各種業務（設計・評価・購買・管理・申請・承認）を一元化したWebプラットフォームを構築
  • API連携により、各システム間のデータを自動同期・転記不要
  • 申請・承認・帳票作成・進捗管理などを自動化し、手作業・重複作業を排除
  • AI/ノーコードツールで現場主導の業務自動化・効率化アプリを迅速に開発・展開
  • ユーザーごとに最適化されたダッシュボードで必要な情報・タスクを一元管理
